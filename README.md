# When Can We Go?

A tiny, no-login group trip availability planner. Share one link; everyone
marks the days they can **definitely** or **maybe** go, from any phone or
laptop. A live heatmap overlays everyone's calendars and highlights the best
windows where the whole group can come.

- No accounts, no app — just open the link.
- Everyone can add and edit availability (built for groups who trust each other).
- Data syncs across devices via a free JSON store; the room lives in the URL hash.

Single self-contained `index.html`. Served via GitHub Pages.
